The MenuCraft app automates the process of creating and publishing daily lunch menus for restaurants, cafeterias, and food service establishments in general.
Even today, it is still common practice for the operations manager or head chef to manually compile lunch menus on a weekly or daily routine basis.
This app aims to automate and streamline this process.
The MVP of the MenuCraft application offers basic functionality:
For each workday of the week, the application generates main courses and soups.
Then allows user to edit the menu and publish it.
Another key feature is the simple management of the library of individual dishes.
